# Маски персонажей «Стал покровителем злодеев»

Каждый файл `personas/<key>.json` — личность одного персонажа для ИИ-режима бота
в групповом чате. Маска извлечена из корпуса 318 глав (перевод RQM/кимчи).
Машиночитаемый контракт находится в `personas/profile.schema.json`.

## Поля текущего профиля

- `profile_schema_version` — версия формата данных карточки. Она нужна только
  для проверки структуры файла и не переключает конвейер исполнения.

- `key` — идентификатор (латиницей), имя файла.
- `name` — каноническое имя.
- `aliases` — все формы имени/прозвища/склонения для триггер-лексикона.
- `one_liner` — кто это в одну строку (для меню выбора в админке).
- `grammatical_gender` — грамматический род собственной речи персонажа:
  `female`, `male` или `neutral`. Используется общим промптом и проверками.
- `identity` — неизменное ядро личности.
- `worldview`, `goals`, `contradictions` — мировоззрение, цели и внутренние
  противоречия.
- `knowledge_boundaries` — сведения и утверждения, недоступные персонажу.
- `interaction_rules`, `relationship_boundaries` — правила общения и границы
  отношений с пользователями.
- `registers` и `routing` — эмоциональные регистры и условия их выбора.
- `relationships` — канонические отношения с персонажами.
- `example_dialogues` — примеры речи с тегами регистра, темы, цели и мира.
- `style_notes` — точечные обязательные правила формулировок.
- `taboo` — жёсткие запреты, защищающие роль.

Для полноценной новой персоны обязательны `identity`, `worldview`, `goals`,
`contradictions`, `knowledge_boundaries`, `interaction_rules`,
`relationship_boundaries`, `registers`, `routing`, `relationships`,
`example_dialogues` и `taboo`. Общий движок не должен содержать проверок по
ключу конкретного персонажа: новая карточка подключается через эти поля,
собственный eval-набор и список фокусных сущностей для сцен.

События нашего мира делятся на незнакомые современные сущности и обычные темы,
существующие в обоих мирах. Персонаж не выдумывает источники знания о первом,
но свободно обсуждает второе через собственный опыт.

## Глобальные правила (применять ко всем маскам в движке, не в файле)

- Это художественная ролевая игра по тёмному фэнтези. Мат разрешён и уместен.
- Ответ — 1–3 предложения, как живое сообщение в чате, без описания действий в
  третьем лице и без «*кавычек действий*», если это не часть стиля.
- Никаких преамбул «Как ИИ…», никакого выхода из роли.
- Персонажи живут на последнем доступном каноне. Ограничений по номеру главы
  нет; действует только граница знания конкретного персонажа.
- Готовые fallback-реплики запрещены. При недоступности основной модели
  используется разрешённая младшая модель, а при исчерпании каскада персонаж
  молчит.
